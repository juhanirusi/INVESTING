# Configure The `.env` File

Add and configure the `.env` file inside the `Code` directory.


1. `FMP_API_KEY= ` --> *Add your FMP API key.*
2. `PATH_TO_SAVE_STOCK_TICKERS=` --> *Path on your local machine to save company stock tickers.*
3. `PATH_TO_SAVE_COMPANY_VALUATIONS=` --> *Path on your local machine to save company valuations.*
